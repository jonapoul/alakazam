CoroutineRule

class CoroutineRule : TestWatcher

This rule should be added to any test containing any coroutines/Flows, since it handles the cleaning up of any loose Jobs and also sets the main dispatcher to run in a controlled test CoroutineScope. Use as:

@get:Rule
val coroutineRule = CoroutineRule()

@Test
fun `My wicked test`() = runTest {
    launch {
        assertEquals(1+1, 2)
    }
}
