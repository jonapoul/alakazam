fun ByteArray.toMacAddress(separator: String = ":"): String
Content copied to clipboard

Converts the ByteArray to a string MAC address representation, e.g. "AB:CD:EF:12:34:56". This also accepts an optional separator parameter, in case you want hyphens or dots between each MAC byte instead of colons.
