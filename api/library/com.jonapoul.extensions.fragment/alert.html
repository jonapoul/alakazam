A different API for showing a dialog window. All configuration is passed in via the config lambda block.

This will also include a single button saying "OK" which does nothing except close the dialog, which can be overridden by passing a call to MaterialAlertDialogBuilder.setPositiveButton as part of the config code block.

Used as:

    alert {
        setTitle("Title")
        setMessage("Whatever")
    }
