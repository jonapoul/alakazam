launchInfiniteLoop

fun CoroutineScope.launchInfiniteLoop(periodMs: Long, delayType: LoopDelayType = LoopDelayType.DELAY_AFTER, dispatcher: CoroutineDispatcher = Dispatchers.Main, skipIf: () -> Boolean? = null, call: suspend () -> Unit): Job

Runs an infinite loop of periodic function calls, scoped to the lifecycle. Takes a LoopDelayType parameter, which specifies whether the periodic delay is placed before the first callback or after. This will only matter on the first run through. The skipIf lambda will be run directly before invoking the main call, to verify whether the call should be run.
