DarkenedBackgroundAlertDialogBuilder

open class DarkenedBackgroundAlertDialogBuilder(context: Context, overrideThemeResId: Int = 0) : MaterialAlertDialogBuilder

Simple wrapper class to apply a darkened background to any alert dialog popups. Note that this doesn't apply to preference dialogs.
