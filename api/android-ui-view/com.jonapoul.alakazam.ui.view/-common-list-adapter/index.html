CommonListAdapter

abstract class CommonListAdapter<T : Any, VH : CommonViewHolder<T>>(diffCallback: DiffUtil.ItemCallback<T>) : ListAdapter<T, VH>

Gives a base implementation of the ListAdapter, which can be used in place of a RecyclerView.Adapter class to reduce boilerplate and give smoother animations when submitting new data to the view.
