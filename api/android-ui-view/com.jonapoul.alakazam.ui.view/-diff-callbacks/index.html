DiffCallbacks

object DiffCallbacks

Allows for easier declaration of DiffUtil.ItemCallbacks for use in androidx.recyclerview.widget.ListAdapter class instances.

DiffCallbacks.basic lets us use the contents of a data class to check equality of two list items, refreshing the view holder if the two are not exactly equal.

DiffCallbacks.withItemId lets us use a lambda method to fetch some uniqueness identifier from the list type. View holders will only be refreshed if this ID is different.
