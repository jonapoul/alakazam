setSimpleNegativeButton

fun <T : AlertDialog.Builder> T.setSimpleNegativeButton(@StringRes text: Int = android.R.string.cancel, onClick: () -> Unit? = null): T

Gives a best-case API to allow callbacks on pressing the "negative" button in a dialog. Here we apply a default (system-translated) string of "CANCEL" and invoke the supplied onClick when it's pressed. If onClick is null, the button will simply close the dialog.
